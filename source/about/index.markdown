## Project Description

**Our topic is :Analysis of Operator Mailing Lists with Real-time Network Measurements.**

This project asks us to use NLP techniques to extract relevant information from the emails to spur real time measurements of the incident in the mailing list (e.g., traceroutes using RIPE Atlas). Reference Paper <a href="http://zhuguanyu.github.io/fundamental_of_network/documents/PAM2015.pdf" style="color:blue">Here</a>.
